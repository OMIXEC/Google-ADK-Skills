# View Examples

Display available example agents, project templates, and usage patterns.

## Usage

```bash
/adk:examples
/adk:examples adk-simple-agents
/adk:examples adk-multi-agent-workflows
```

## Content

Shows:
- Available agent templates
- Example implementations
- Project structure
- Common patterns
- Quick start guides

Use examples as starting points for your own agents.
