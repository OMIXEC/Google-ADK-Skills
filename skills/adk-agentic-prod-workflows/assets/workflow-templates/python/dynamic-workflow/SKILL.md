# Dynamic Workflow (Python)

Programmatic workflow where Python code drives agent execution order. No graph DSL — full control via async/await.

## When to Use

- Complex branching that graphs can't express cleanly
- Loops with dynamic iteration counts
- Conditional agent selection at runtime
- Workflows that need external state machines
