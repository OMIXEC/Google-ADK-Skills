# Collaborative Workflow (Python)

Coordinator agent routes tasks to specialized sub-agents. Each sub-agent has its own tools and instructions.

## When to Use

- Multi-domain support (billing, tech, general)
- Tasks requiring different expertise
- Research → Code → Review pipelines
- Human-in-the-loop with escalation paths
