## Run Instructions

### Running Ralph (Autonomous Development)

Ralph will automatically execute the task list from `.ralph/@fix_plan.md`.

```bash
cd /home/omixec/Claude-ADK-Skills

# Start Ralph autonomous development
python -m ralph --prompt .ralph/PROMPT.md --plan .ralph/@fix_plan.md

# OR with options:
python -m ralph \
  --prompt .ralph/PROMPT.md \
  --plan .ralph/@fix_plan.md \
  --log-dir .ralph/logs \
  --max-loops 50
```

### Ralph Workflow

Each Ralph loop:
1. **Read Fix Plan**: Finds next unchecked task in `.ralph/@fix_plan.md`
2. **Execute**: Performs task step-by-step (create files, write code, run tests)
3. **Report**: Outputs status in standard format
4. **Update Plan**: Marks completed tasks as [x]
5. **Loop**: Continues until all tasks complete or EXIT_SIGNAL triggered

### Monitoring Ralph Progress

```bash
# Watch logs in real-time
tail -f .ralph/logs/latest.log

# Check progress
cat .ralph/@fix_plan.md | grep "^\- \[x\]" | wc -l

# View generated documentation
ls -la .ralph/docs/generated/
```

### Stopping Ralph

```bash
# Graceful stop (waits for current task)
Ctrl+C

# Force stop (immediate)
Ctrl+Z

# Check if Ralph is still running
ps aux | grep ralph
```

### Ralph-Driven Development

Ralph handles most development. The workflow is:

1. **Specification**: Ralph reads `.ralph/specs/plugin-requirements.md`
