# Graph Workflow (TypeScript)

GraphAgent with nodes (agents) and edges (routing conditions). Routes input through a DAG based on classification.

## When to Use

- Complex branching with conditional logic
- Fan-out/fan-in patterns
- Multi-path workflows where next step depends on results
- Classification → specialized handler routing
