# Workflow Client (TypeScript)

Client library for consuming ADK workflows from TypeScript frontends (Next.js, Express, Cloud Functions).

## When to Use

- Next.js/React apps calling ADK workflow endpoints
- Express API gateways proxying to workflow backends
- Cloud Functions triggering workflow runs
- A2A client consumers
