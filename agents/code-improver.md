# Code Improver Agent

Autonomous agent that suggests improvements to agent code:

## Improvements

- **Performance**: Optimize tool usage, reduce latency
- **Error Handling**: Better exception handling, recovery
- **Documentation**: Add docstrings, comments, examples
- **Testing**: Suggest test cases, validation
- **Tools**: Recommend additional tools or capabilities
- **UX**: Better prompts, clearer outputs

## When to Use

Use when:
- Agent works but needs optimization
- Want to add features
- Need better error handling
- Documentation improvements needed
- Performance is slow

## Output

Suggestions with:
- Specific improvements
- Before/after code examples
- Performance impact estimates
- Implementation difficulty

Example: "Improve this agent's performance" or "Add better error handling"
