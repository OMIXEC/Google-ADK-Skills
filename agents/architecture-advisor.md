# Architecture Advisor Agent

Autonomous agent that guides architecture decisions:

## Guidance

- **Patterns**: Recommends orchestration pattern (supervisor, hierarchical, etc.)
- **Tools**: Suggests tools and integrations
- **Memory**: Advises on memory system design
- **Scaling**: Design for expected load
- **Integration**: Recommends external services
- **Trade-offs**: Explains complexity vs. capability trade-offs

## When to Use

Use when:
- Designing multi-agent system
- Uncertain about approach
- Need architecture review
- Want pattern recommendations
- Need trade-off analysis

## Output

Recommendations with:
- Best pattern for use case
- Architectural diagram (text)
- Component breakdown
- Trade-off analysis
- Implementation roadmap

Example: "Design a multi-agent research system" or "Should I use supervisor or hierarchical pattern?"
